# Scope

## Core Features
- Landing page with rotating intro
- CV page with PDF export
- Contact page with anti-spam (honeypot + timing + Turnstile)
- Projects page (GitHub repos)
- Travels page with interactive map and statistics
- Photos page with Instagram galleries and lightbox viewer
- Google OAuth login (admin access)
- Container-based local dev (docker-compose)

## Admin Tools
- Trip management (add/edit/delete with destinations, cities, participants)
- Instagram post labeler (keyboard-driven workflow for categorizing posts)
- Location check-in
- Close ones management

## Production
- GCP Cloud Run deployment
- Cloud SQL database
- GCS photo storage with CDN
